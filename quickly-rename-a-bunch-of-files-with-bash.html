Quickly Rename a Bunch Of Files With Bash

Sometimes you might feel the need to rename a bunch of files from one file extension to another.

This can easily be done with built in shell functions. To quickly rename a bunch of file endings from say for example *.markdown to *.md you simply can execute the following (bash) command.

for i in *.markdown; do mv $i "${i%.markdown}.md"; done
